Proje Adı: Aetheris Portfolio PWA

Proje Amacı: github API kullanarak microsofttan veri çekerek yazılım şirketi portföy sitesi

Kullanılan API: GitHub REST API (Link: https://docs.github.com/en/rest)

CSS Framework: Bootstrap 5

PWA Özellikleri:

manifest.json ile kurulabilirlik sağlandı.

Service Worker ile App Shell önbelleklendi.

Bağlantı koptuğunda offline.html sayfası devreye girer.
